    An intentionally simple templating language. This function is used to swap out event names when binding with mapEvents. Replaces instances of {value} given an object of properties. If these values are absent, nothing is replaced

    For example:

    template('{foo}', { foo: 'bar' }) //=> 'bar'
template('{foo}') //=> '{foo}'
